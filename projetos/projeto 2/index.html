Banco de Dados de Locação de Veículos

Este banco de dados foi desenvolvido para uma locadora de veículos e organiza as informações principais de maneira eficiente. Ele inclui tabelas para Lojas, Clientes, Automóveis e Aluguel, facilitando o gerenciamento de todos os aspectos da locação de veículos. Neste projeto foi usado o Microsoft Access onde eu aprendi a criar uma base de dados capaz de suprir demandas de falta de informações internas em uma empresa.
